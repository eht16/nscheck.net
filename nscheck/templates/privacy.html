Server log files

The website operator collects data about access to the site and stores it
as "server log files". The following data is logged:

  • Visited website
  • Time of access
  • Amount of data sent in bytes
  • Source / reference from which you came to the page
  • Used browser
  • Used operating system
  • Used IP address

This data is only stored for error detection and debugging purposes. There is no evaluation or linkage of this data with other sources.
Server log files get deleted automatically after 60 days.

Cookies

This website sets one single cookie named "csrf_token".
The "CSRF" protection cookie helps to prevent Cross-site request forgery attacks and expires at the end of the browser session (i.e. when the browser is closed). This cookie does not contain any personal data but only a randomly generated character string.
